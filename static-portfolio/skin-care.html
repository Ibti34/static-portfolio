Skincare E-commerce Platform

A modern web shop dedicated to premium beauty products, featuring a full shopping experience, secure authentication, and product management.

Overview

This platform was built to demonstrate a complete B2C e-commerce workflow. It handles everything from user registration and secure login to browsing product catalogs with high-quality images. The system is designed with a "mobile-first" approach to ensure beauty shoppers have a seamless experience on any device.

Key Features

  • • Secure User Authentication
  • • Product Listing & Categories
  • • Shopping Cart Logic
  • • Image Upload & Storage
  • • Responsive Admin Dashboard
